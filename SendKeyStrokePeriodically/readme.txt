SendKeyStrokePeriodically.pkg
-----------------------------
It will:
 - install AppleScript to the /Library/SendKeyStrokeToVerizonClient/
 - install com.send-keystroke-periodically-to-verizon-client.plist to /Library/LaunchAgents/
 - launch installed script to run periodically
 

Uninstall
---------

Double click on Uninstall_Send_KeyCodePeriodically.tool file.

In appeared Terminal window enter sudo password.
